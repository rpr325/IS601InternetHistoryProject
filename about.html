The History of the Internet Project

About

The History of the Internet project is geared toward reinforcing the use of HTML and the ability to collaborate work done in Webstorm using the Git platform. The project focuses on the summary of five main video components: The History Channel's "The Invention of the Internet", The True Story of the Internet - Browser Wars Part 1, The True Story of the Internet- Internet Search Part 2, The True Story of the Internet- Dot Com Bubble Part 3, The True Story of the Internet- People Part 4. This was all configured using HTML to create a website titled "History of Internet Website"
